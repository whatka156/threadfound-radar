Enhanced mobile/PWA version.

Upload/replace these files in the root of your GitHub repo:
- index.html
- manifest.json
- service-worker.js
- icon-180.png
- icon-192.png
- icon-512.png
- apple-touch-icon.png

Then open the GitHub Pages URL in Safari, delete the old Home Screen shortcut if needed,
and add it again using Share > Add to Home Screen.

- standalone/full-screen PWA settings
- improved app icon
- loading/splash screen
- offline app shell
- sync saved/syncing/offline indicator
- smoother mobile styling
